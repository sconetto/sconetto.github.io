RaDop - Radar de Efeito Doppler para prevenção de acidentes

  • Linguagem: Python/JavaScript
  • Papel: Gerente de Software/Desenvolvedor

O RaDop é um produto de engenharia com sistemas de business intelligence e manutenção de radares construídos com antenas de efeito Doppler para prevenção de acidentes em estradas sinuosas e de baixa visibilidade. A solução de software se deu numa arquitetura microsserviços com um Dashboard para gestores, um app Android para mantenedores e uma stack de serviços e funções para estes ambientes.

SAPL - Sistema de Apoio ao Processo Legislativo

  • Linguagem: Python Django
  • Papel: Desenvolvedor

O SAPL é um ferramenta que visa auxiliar o Processo Legislativo (PL), sendo este o conjunto de atividades de uma Casa Legislativa. É manutenido pelo Interlegis com auxílio da comunidade do GITEC. O sistema foi imaginado para automatizar o PL visando os benefícios relacionados com a qualidade dos serviços prestados pelas Casas e com o trabalho dos parlamentares.
